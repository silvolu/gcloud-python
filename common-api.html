gcloud.credentials.get_for_service_account(client_email, private_key_path, scope=None)[source]¶

Gets the credentials for a service account.

Note

You should not need to use this function directly. Instead, use the helper methods provided in gcloud.datastore.__init__.get_connection() and gcloud.datastore.__init__.get_dataset() which use this method under the hood.

Parameters:
  • client_email (string) – The e-mail attached to the service account.
  • private_key_path (string) – The path to a private key file (this file was given to you when you created the service account).
  • scope (string or tuple of strings) – The scope against which to authenticate. (Different services require different scopes, check the documentation for which scope is required for the different levels of access to any particular API.)
Return type:

oauth2client.client.SignedJwtAssertionCredentials
Returns:

A new SignedJwtAssertionCredentials instance with the needed service account settings.
